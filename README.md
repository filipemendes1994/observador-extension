# observador-extension
A Chrome extension to unlock premium articles and block ads on Portuguese news website, Observador.

## Project setup

### Requirements

- Node.js >= 6.0

#### Note:
Yarn isn't required, although it's used to install all project dependencies and run scripts. If you have any question, check the full [documentation](https://yarnpkg.com/lang/en/docs/cli/install/).

### Install dependencies

```sh
yarn
```
